A modern, full-featured Point of Sale (POS) desktop application designed for ice cream shops and similar retail businesses. Built with Python and tkinter, it features a premium dark-themed UI, real-time sales tracking, inventory management, and a Windows installer.

---

## Features

### Sales & Cart
- Fast product search and add-to-cart from a tree view
- Real-time cart sidebar with quantity editing and item removal
- Multiple payment methods (cash, card, transfer)
- Automatic change calculation
- Invoice number generation per transaction

### Product & Inventory Management
- Create, edit, and delete products with categories and pricing
- Stock tracking with visual alerts (low stock / out of stock)
- Stock movement history
- Import/export to Excel and CSV

### Orders (Pedidos)
- Create and manage supplier orders
- Mark individual items as received
- Track pending and completed orders

### Reporting & Analytics
- Daily, weekly, and monthly sales reports
- Revenue charts using Matplotlib
- Export reports to PDF (via ReportLab) and Excel
- Inventory and order summaries

### User Management & Authentication
- Role-based access control: `admin` and `cajero` (cashier)
- bcrypt password hashing (12 rounds)
- Login attempt tracking and lockout protection
- Session logging

### Configuration
- Business name, tax rate, currency symbol
- Brevo (Sendinblue) email integration for sending invoices
- Configurable via `config.json`

### Windows Installer
- 6-step graphical wizard (`setup.pyw`)
- License agreement, install path selector, optional shortcuts
- Registers the app in Windows Add/Remove Programs
- Creates Desktop and Start Menu shortcuts
- Bundled as a standalone `.exe` via PyInstaller (no Python required on target machine)

## Project Structure

```
app/
├── main.py            # Main application — UI and all screens
├── database.py        # SQLite manager (Singleton, thread-safe)
├── services.py        # Business logic layer (Auth, Products, Sales)
├── models.py          # Data models (Usuario, Producto, Venta, Pedido…)
├── utils.py           # Helpers: logging, config, formatting, validation
├── config.json        # Runtime configuration (business name, tax, etc.)
├── requirements.txt   # Python dependencies
└── tests/             # Unit and integration tests (pytest)

setup.pyw              # Windows installer wizard (6 steps)
landing/
└── index.html         # Marketing landing page (dark theme, single file)
dist/
└── TPVElite_Setup.exe # Prebuilt installer (generated by PyInstaller)
```

## Getting Started

### Requirements

- Python 3.8 or higher
- pip

### Install dependencies

```bash
cd app
pip install -r requirements.txt
```

### Run the application

```bash
python app/main.py
```

Default admin credentials:
- **Username:** `admin`
- **Password:** `admin123`

> Change the admin password immediately after first login.

---

## Building the Installer

Requires [PyInstaller](https://pyinstaller.org/):

```bash
pip install pyinstaller
python -m PyInstaller --onefile --noconsole --name TPVElite_Setup --add-data "app;app" setup.pyw
```

The output will be at `dist/TPVElite_Setup.exe`.
